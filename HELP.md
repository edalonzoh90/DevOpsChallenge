## Notes
__Create a ssh key__  
`ssh-keygen -f mykey`  

__Connect to ec2 via ssh__  
`ssh 3.84.92.132 -l ubuntu -i ~/Documents/ssh/mykey`

__Install httpd-apache2__  
`sudo apt-get update  
sudo apt install apache2 -y`  

__Install node__  
`npm install`  

__Run node app__  
`node app.js`  

__Ports in use__  
`netstat -tnlp`
